Welcome to my cybersecurity learning repository! This space is dedicated to documenting my daily study logs, technical notes, hands-on lab exercises, and Capture The Flag (CTF) challenge writeups as I build a solid foundation in Cyber Security, Networking, and System Security.

## 🎯 Primary Goals & Objectives
* **Daily Consistency:** Dedicating a minimum of 1 hour daily to structured cybersecurity learning.
* **Hands-on Practice:** Solving beginner-to-intermediate CTF challenges on platforms like PicoCTF, TryHackMe, and Cylab.
* **Skill Building:** Mastering core concepts in Linux Administration, Computer Networking, Web Security, and Cryptography.
* **Documentation:** Maintaining a professional learning log and clear writeups for future reference.

## 📚 Core Learning Domains
### 1. 🐧 Linux Fundamentals & CLI
- Terminal Navigation (`ls`, `cd`, `pwd`, `find`, `grep`)
- File Permissions & User Access Management (`chmod`, `chown`, `sudo`)
- Basic Shell Scripting & Command Chaining (`|`, `>`, `>>`)
### 2. 🌐 Networking Essentials
- OSI Model & TCP/IP Protocol Suite
- Network Protocols & Port Numbers (HTTP/HTTPS, SSH, FTP, DNS)
- Traffic Inspection & Packet Analysis Basics
### 3. 🔐 Cryptography & Steganography
- Base Encodings & Ciphers (Base64, Hex, ASCII, Caesar / Rot13)
- Hash Functions (MD5, SHA-256)
- File Metadata Analysis & Hidden Data Extraction
### 4. 💻 Web Application Security
- HTTP Request & Response Headers
- Browser Developer Tools (Inspect Element, Network Tab, Storage/Cookies)
- Basic Understanding of Web Vulnerabilities (OWASP Top 10 Concepts)
